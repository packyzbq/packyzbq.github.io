CNCF周报-2-25

Posted on February 25, 2020

要闻

ETCD 3.4.4 发布

这个版本 ETCD 主要修复了一个在碎片整理时崩溃的bug（https://github.com/etcd-io/etcd/pull/11613）

其他：

  • 新增了几个 etcd_debugging_* 的 metrics endpoint
  • 修复了两个认证方面的bug (issue#11414)

release note(https://github.com/etcd-io/etcd/blob/master/CHANGELOG-3.4.md)

jager 1.17.0 发布

这个版本，jager 增加了一些支持kafka 的特性：

  • 在命令行中，增加 kafka batch 相关配置，包括 suffixBatchSize suffixBatchLinger BatchMaxMessages
  • 为kafka 存储增加了基础认证

此外也允许在Jaeger Query HTTP API上配置其他标头 #2056

release note(https://github.com/jaegertracing/jaeger/releases/tag/v1.17.0)

istio 1.5.0 架构更改

在 istio 1.4 及以前版本，其架构中由多个模块组成，包括 polit、galley 等，但是在 1.5.0 版本中，其架构发生重大变化，由多个组件的形式转变为 istiod 和 istio-agent。

详见文章

上游信息

  • #88292 kubectl 以前的 --dry-run-server参数被 --dry-run=server|client|none取代
  • #87975 kubeadm upgrade node config 从 v1.15 起标记废弃，现在正式移除，请使用 kubeadm upgrade node phase kubelet-config 代替。
