New Start

Posted on February 16, 2019

本站的第一篇博文，就先回顾下过往随意写写一下吧。

鄙人作为一名程序员从本科开始到现在也有七八年的时间了，居然连一个属于自己的博客都没有，我也是有点瞧不起自己啊。一方面是自己懒，不想写文章更不想打理；另一方面也是自己肚子里确实没有什么墨水，一直认为没有什么好写的，既没有大佬们不明觉厉的学术研究和高技术含量的项目，也不想只是划划水记录下无聊的生活。但是回想下这几年，在不同的领域里大大小小的项目也都折腾过几个，搞过嵌入式、研究过分布式系统，机器学习和数据挖掘也浅尝辄止，最终落在了java后台这样偏工程的工作，虽然能够折腾各种框架，搞起来一个项目也蛮有成就感，但还是觉得缺少点什么，总是觉得这些东西对我来说都是黑盒，我只是负责组装，可能还需要在性能上打磨，久而久之思维也就固化了。现在开一个自己的博客，也是想记录下平时项目里值得记录的技术点或者实现细节(包括踩的坑也有可能)，其次也是为了鼓励自己多多思考吧，不要只在核心技术外围徘徊，对于一些优秀的框架还是需要研究研究其内部细节，开放自己的思维。当然，我还是要维护自己类wiki的知识库的，不过那些就保存在Notion里咯。

新的一年也开始了，在这里给自己立几个flag吧，不说克服自己的拖延症，至少能鞭笞下自己吧：

迭代自适应推题的逻辑(这是大头)，并多多记录对自适应推题的想法
redis，kafka内部原理
搞搞算法
维护个人CMS
当然还有练好我的小口琴
纸上得来终觉浅，技术上的东西不能只知其然，遇到自己感兴趣的还是需要努力实践滴，不要怕麻烦，尽量不拖延！总之，加油！
